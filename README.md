# ExpoAir Monorepo Structure

ExpoAir is a project that consists of several components, each serving a specific purpose. Here's a brief overview of         
the main folders and their respective functions:

- **firmware**: Contains Arduino/ESP32 code for IoT devices.
- **backend**: A Python FastAPI server powering the API endpoints of ExpoAir.
- **ml**: Jupyter notebooks and scripts for data analysis, machine learning models, and processing.
- **mobile**: A Flutter application developed using Dart for mobile device interaction.
- **dashboard**: Streamlit application for data visualization and monitoring.

Each folder contains related files and code necessary to develop and deploy the ExpoAir system effectively.
